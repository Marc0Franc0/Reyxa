# Tutorial para instalar y utilizar Reyxa en tu computadora

IMPORTANTE: Mencionar que el tutorial aplica en computadoras con el sistema operativo Windows 10 y Windows 11.


1) Descargar el editor de texto Visual Studio Code
Link: https://code.visualstudio.com/download

2) Descargar Java y JDK
Link JAVA: https://www.java.com/es/download/ie_manual.jsp
Link JDK: https://www.oracle.com/java/technologies/downloads/
TUTORIAL DE INSTALACIÓN DEL JDK: https://www.youtube.com/watch?v=oAin-q1oTDw


3) Descargar Angular. La versión utilizada en el proyecto es la 13.3.6
Link Angular 13.3.6: https://www.npmjs.com/package/@angular/common/v/13.3.6
TUTORIAL DE INSTALACIÓN DE ANGULAR: https://www.youtube.com/watch?v=6741ceWzsKQ

4) Descargar el proyecto (Reyxa) desde Github
Link: https://github.com/Marc0Franc0/Reyxa

5) Una vez tengamos todos lo necesario descargado e instalado, procederemos a abrir el Visual Studio Code. Posteriormente nos dirigiremos a la opción superior izquierda que dice "Archivo", luego a la opción que dice "Abrir Carpeta". Una vez que clickeemos allí, nos abrirá el exlorador de Windows donde deberémos buscar la carpeta llamada Reyxa, es decir, la carpeta de que descargamos de GitHub que contiene el proyecto en su totalidad.
